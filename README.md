# Gaming History

Minha jornada gamer - Todos os jogos que joguei desde o Atari ate hoje.

## Plataformas

- Atari
- MS-DOS
- Dynavision
- Super Famicom
- PlayStation 1 / 2 / 3 / 4 / 5
- Xbox 360
- PC
- Switch 2

## Funcionalidades da Pagina

- Busca por nome, genero, plataforma
- Filtros por plataforma, genero e status
- Ordenacao por nome, ano, plataforma
- Visualizacao em grade ou lista
- Modal com detalhes de cada jogo
- Exportar lista em CSV
- Modo claro / escuro
- Linha do tempo com eras dos jogos
- Estatisticas gerais
